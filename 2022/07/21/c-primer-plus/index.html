C语言基础

基本数据类型

关键字

基本数据类型由十一个关键字组成：int、long、short、unsigned、char、float、double、signed、_Bool、_Complex、_Imaginary。

有符号整形

有符号整形可以表示正整数和负整数。

  • int——系统给定的基本整数类型。C语言规定int类型不小于16位。
  • short或short int——最大的short类型整数不大于最大的int类型整数。C语言规定short类型至少占16位。
  • long或long int——该类型可表示整数不小于最大的int类型整数。C语言规定long类型至少占32位。
    Q：什么情况下用long类型变量代替int类型变量？
    A：在系统中要表示的数超过了int的范围；在处理更大的数时，需要一种在所有系统上都保证至少是32位的类型，以提高程序的可移植性。
  • long long或long long int——该类型可表示整数不小于最大的long类型整数。long long 类型至少占64位。
    一般而言，long类型占用内存比short类型大，int类型宽度要么和short类型相同，要么和long类型相同

无符号整形

无符号整形只能用于表示非负数，因此无符号整形比有符号整形表示的正整数范围更大，比如int类型表示的范围是-2147483648 ~ 2147483647 ( −231∼231−1-2^{31} \sim 2^{31} -1−231∼231−1 ) ，而unsigned int表示的范围是0 ~ 4294967295 ( 0∼232−10 \sim 2^{32} -10∼232−1 ) 。在整形类型前加上关键字unsigned可表示该类型为无符号整形：unsigned int、unsigned long、unsigned short。单独的 unsigned 用于表示 unsigned int。

字符类型

可打印出来的符号都是字符（如A、&、+ 等）。根据定义，char类型表示1个字符要占用1个字节的内存。1字节通常是8位，但是如果要表示基本字符集，也可以是16位或更大。

  • char——字符类型的关键字。有些编译器使用有符号的char，有些则使用无符号的char。在需要时，可在char前加上关键字unsigned或signed指明使用何种类型。

布尔类型

布尔值表示true和false。C语言用1表示true，0表示false。

  • Bool——布尔类型的关键字。布尔类型是无符号int类型，所占用的空间只要能储存0或1即可。

实浮点类型

实浮点类型可表示正浮点数和负浮点数。

  • float——系统的基本浮点类型，可精确表示至少6位有效数字。
  • double——储存浮点数的范围更大，能表示比float类型更多的有效数字（至少10位，一般更多）和更大的指数。
  • long double——储存浮点数的范围比double更大，能表示比double更多的有效数字和更大的指数。

复数和虚数浮点数

虚数类型是可选的类型。复数的实部和虚部类型都基于实浮点类型来构成。

  • float _Complex
  • double _Complex
  • long double _Complex
  • float _Imaginary
  • double _Imaginary
  • long double _Imaginary

字符串和格式化输入、输出

sizeof()和strlen()函数

sizeof() 是以字节为单位给出对象的大小，而 strlen()函数则给出字符串中字符长度。例如 a[20]=“Hello world!” 通过 sizeof() 函数得出的结果是20，是此数组的大小，而 strlen() 函数得出的结果是12，也就是此字符串的长度。
另外，当 sizeof()接受的参数为类型时，需要使用圆括号，如 sizeof(int)，当接收参数为特定量时，可以不加圆括号，如 sizeof a。

常量和C预处理器

预处理器除了使用 #include 来包含其他文件信息，还可以用 #define 来定义常量，格式是：显示 #include 符号变量名 符号常量值。如：#define Name value。
该指令还可以用来定义字符和字符串常量，前者用单引号，后者用双引号，如

  • #define BEEP ‘\a’
  • #define TEE ‘T’
  • #define ESC ‘\033’
  • #define OOPS “Now you have done it!”

值得注意的是，如果定义的符号常量是一个表达式，则在预处理时会先将表达式代入再进行计算。如 #define ADD 2+3 如果有这样一个表达式：$ 2ADD $，其进行的计算为 $ 22+3 $，而非 $ 2*(2+3) $，如要达到后者的效果，需如下定义：#define ADD (2+3)。

printf()和scanf()

转换说明及其打印的输出结果

转换说明 输出 转换说明 输出
%a 浮点数、十六进制和p计数法 %A 浮点数、十六进制和p计数法
%e 浮点数，e计数法 %E 浮点数，E计数法
%c 单个字符 %d 有符号十进制整数
%f 浮点数，十进制计数法 %p 指针
%g 根据值得不同，自动选择%F或%e。%e格式用于指数小于-4或者大于或等于精度时 %G 根据值得不同，自动选择%F或%E。%E格式用于指数小于-4或者大于或等于精度时
%i 有符号十进制整数（与%d相同） %o 无符号八进制整数
%s 字符串 %u 无符号十进制整数
%x 无符号十六进制整数，使用十六进制数 0f %X 无符号十六进制整数，使用十六进制 0F

printf()中的标记

标记 含义
- 从字段左侧开始打印。示例"%-20s"
+ 显示打印的有符号值的符号。示例"%+6.2f
空格 有符号值如果为正，则值前显示空格；如果为负，则值前显示-号+标记覆盖的空格。示例"% 6.2f"
# 把结果转换为另一种形式。如果是%o格式，则以0开始；如果是%x或%X格式，则以0x或0X开始；对于浮点格式，即使后面没有任何数字，也打印一个小数点字符；对于%g和%G格式，防止结果后面的0被删除。示例"%#o"、“%#8.0f”、“%+#10.3e”
0 对于数值格式，用前导0代替空格填充字段宽度。对于整数格式，如果出现-标记或指定精度，则忽略该标记。示例"%010d"、“%0.83f”
